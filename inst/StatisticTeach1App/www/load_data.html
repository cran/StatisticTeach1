User Guide: Load Data Module

This module allows you to import and configure datasets for your statistical analysis. Follow these simple steps to load your data into the application:

1. Data Source Configuration

  • Use Example Data: Check the "Use example data set?" box if you want to explore the tool using pre-loaded data.
  • File Upload: To use your own dataset:
    • Click the Browse... button to select a file from your computer.
    • If your file has multiple sheets, select the correct one from the Worksheet Name dropdown menu.
    • Click the blue Load Data button to import the file.

2. Data Preview

  • Once the data is loaded, it will be displayed in the large Data Preview panel on the right side of the screen. Use this panel to verify that your information was imported correctly.

3. Column Types Configuration

  • To ensure accurate statistical calculations, define the correct data types for your variables in this section.
  • Select the formats and click the green Apply Column Types button to save your settings.
