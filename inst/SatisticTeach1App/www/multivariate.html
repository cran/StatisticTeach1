Multivariate Analysis

The Multivariate module allows you to explore relationships, variances, and correlations among multiple continuous variables simultaneously. Use the sidebar to configure your data before viewing the results.

1. Select Your Variables

  • Use the Variables dropdown menu to select the data columns you wish to analyze.
  • Because this is a multivariate analysis, you can (and should) select multiple variables at the same time from the list.

2. Grouping Data (Optional)

  • If you want to calculate your multivariate statistics separated by specific categories, check the Statistics by group? box and select your grouping variable.

3. View Summaries

  • Navigate to the Summaries tab in the main panel to view your results instantly in interactive tables:
    • Correlation matrix: Displays the correlation coefficients between all possible pairs of your selected variables, helping you identify strong linear relationships.
    • Variances and covariances matrix: Shows the variance of each individual variable along the diagonal, and the covariance between different variables off the diagonal.
